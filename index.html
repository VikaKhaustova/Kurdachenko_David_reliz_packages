Вигідні комплексні набори послуг

«Чисті лапки»

Базовий догляд для підтримання гігієни хвостика.

  • Двофазне купання преміум-шампунем за типом шерсті
  • Двайлива сушка компресором та вичісування
  • Підстригання та підпилювання кігтів
  • Гігієнічна чистка вушок та очей

Ціна (коти / міні-собаки) — 600 грн

Ціна (середні / великі собаки) — від 900 грн

«Повна прокачка»

Популярний вибір для підтримання ідеального модельного вигляду.

  • Увесь комплекс «Чисті лапки»
  • Модельна стрижка або експрес-линька від топ-майстра
  • Чищення зубів спеціальною пастою (мануальне)
  • Фінішне парфумування професійними парфумами

Ціна (коти / міні-собаки) — 950 грн

Ціна (середні / великі собаки) — від 1400 грн

«VIP-Релакс»

Преміум-день для хвостиків, які заслуговують на найкраще.

  • Увесь комплекс «Повна прокачка»
  • Гідромасажна ванна з озонотерапією (20 хвилин)
  • Поживна лікувальна маска для шерсті (грязьова / протеїнова)
  • Парафінотерапія та легкий масаж лапок бальзамом
  • Фотосесія у спеціальній фотозоні салону

Ціна (коти / міні-собаки) — 1600 грн

Ціна (середні / великі собаки) — від 2400 грн

«Ніжні щічки»

Перший адаптаційний візит для малюків (цуценят та кошенят) до 6 місяців.

  • М'яке купання гіпоалергенним дитячим шампунем
  • Знайомство з шумом фену та інструментами в ігровій формі
  • Акуратне підрізання кігтиків без стресу
  • Корисний смаколик у подарунок за хоробрість

Ціна (для всіх малюків) — 400 грн
